Enabling

By default the plugin is not enabled. To enable it, you should right mouse click on the project whose XML files you want handled by the XML Bookmarks plugin and select Configure/Enable XML Bookmarks Nature. This will add the nature and 1 builder to your project configuration.

Disabling

To disable the plugin, you should right mouse click on the project whose XML files you no longer want handled by the XML Bookmarks plugin and select Configure/Disable XML Bookmarks Nature. This will remove the nature and 1 builder from your project configuration.
