Using the add bookmarks editor ruler command

You can create bookmarks in an XML file by using the standard Eclipse add bookmarks (right mouse click on the left margin in the XML editor). When you save the XML file this will define also create the ancillary bookmarks XML file if you have set the create bookmarks file preference.

Creating the bookmarks ancillary file with bookmarks information

You can create in an XML editor or using an external program an ancillary XML file that looks like this: 
<markers>
  <marker>
  	<lineNumber>2</lineNumber>
  	<message>important</message>
  </marker>
  <marker>
  	<lineNumber>12</lineNumber>
  	<message>less important</message>
  </marker>
  <marker>
  	<lineNumber>43</lineNumber>
  	<message>minor important</message>
  </marker>
</markers>
This will create 3 bookmarks with the given text in the message element on the line number specified by the lineNumber element. If you want these bookmarks to be created on file purchase-orders.xml make sure you save the ancillary file as purchase-orders.bookmarks.xml. You can look at the bookmarks using the Eclipse standard Bookmarks view.
