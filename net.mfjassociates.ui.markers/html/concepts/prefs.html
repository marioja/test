Opening the XML Bookmarks preferences

Open the Window/Preferences dialog. In the left categories select XML/XML File/XML Bookmarks

automatically create XML bookmarks files

This preference is either true or false (default). This preference will permit (true) or disallow (false) the creation of ancillary XML bookmarks files when modifying Eclipse bookmarks in XML files. This means that when it is false the ancillary file is not created. If it already exists, it will be updated.
